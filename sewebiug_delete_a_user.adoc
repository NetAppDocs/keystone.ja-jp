= ユーザを削除します
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ユーザを削除するには、このメソッドを使用します。


NOTE: 削除されたユーザを回復することはできません。



=== 手順

. メニューから Administration > Users を選択します。
. リストからユーザを探して、 [ 削除 ] アイコンをクリックします。
. 削除の確認ダイアログボックスで、削除するユーザーの名前を入力します。
. [* 確認 *] をクリックします。
